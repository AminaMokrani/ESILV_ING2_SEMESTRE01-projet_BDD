# ESILV_ING2_SEMESTRE01-projet_BDD
Projet académique réalisé en deuxième année d’école d’ingénieur à l'ESILV, consistant en la conception d'une base de données d'une salle de sport et de gérer la connexion des différents membres.

AFIN DE FACILITER LA PHASE DE TESTE NOUS VOUS PRESENTONS QUELQUES INFOS :

pseudo admin principal : 'adminP'
mdp admin principal : 'P123'

pseudo admin secondaire : 'adminS'
mdp admin secondaire : 'S123'

QUELQUES MEMBRES DEJA VALIDES :

pseudo : 'membre1'
mdp : 'mdp1'
associé à : 'Jean','Martin','jean@gmail.com'

pseudo : 'membre2'
mdp : 'mdp2'
associé à : 'Sara','Dupont','sara@mail.com'

REMARQUES :
- Nous avons mis en place une salle pleine a petite capacité ( salle A - id 1 ) afin de pouvoir tester un cas de salle pleine.
- Il est crucial que nous notiez/reteniez votre id (ou l'id du membre, cours etc que vous souhaitez chercher ou modifier) car notre application sur base 
sur un système de parcours/recherche selon l'id de nos éléments.
Il y'a plusieurs fonctionnalités qui permettent justement d'afficher l'ensemble des éléments (membres, coachs etc) nous vous prions donc de les utiliser 
afin de vous renseigner.
